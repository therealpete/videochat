Erstelle eine Verknüpfung auf deinem Desktop um direkt einen Video-Chat zu starten.
Gib dazu einen möglichst eindeutigen Konferenzraum-Namen ein.
Erlaubt sind Buchstaben und Zahlen, aber keine Sonderzeichen.

Die Verknüpfung wurde erfolgreich erstellt, du kannst nun direkt von deinem Desktop aus den Video-Chat starten.
